# BeaujoLoop Landing Page

This repository contains the initial landing page for BeaujoLoop, a premium vintage scooter concierge service for the Beaujolais region.

## Features
- Dark Luxury aesthetic with deep charcoal background and muted gold accents.
- Responsive single-page design showcasing the brand, fleet, curated routes, and packages.
- Email waitlist form connected to a dedicated agent database for lead capture.

## Technologies
- HTML and CSS, with Google Fonts for typography.
- JavaScript to handle form submission and user feedback.

## Deploy
Deployed on Vercel for fast global delivery and ease of update.
